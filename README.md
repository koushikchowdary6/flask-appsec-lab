# Flask AppSec Lab 🔐

A hands-on web application security lab built with Flask, demonstrating common OWASP Top 10 vulnerabilities and their fixes.

## What This Project Covers

- SQL Injection — vulnerable endpoint vs. parameterized query fix
- Cross-Site Scripting (XSS) — reflected XSS demo and sanitization
- Broken Authentication — weak session handling and secure implementation
- Security Misconfiguration — debug mode risks and hardening
- Insecure Direct Object Reference (IDOR) — access control bypass demo

## Why I Built This

To develop a deeper understanding of how attackers exploit web vulnerabilities and how developers can write more secure code from the ground up — aligned with OWASP Secure SDLC principles.

## Tech Stack

- Python / Flask
- OWASP Top 10 Framework
- Burp Suite (for testing)
- Git / GitHub

## Key Takeaways

- Identified and remediated 5+ vulnerability classes
- Mapped findings to OWASP Top 10 and CWE references
- Documented attack paths and secure coding fixes
